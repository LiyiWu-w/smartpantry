# SmartPantry

A personal recipe and fridge inventory management web application built for college students and young professionals who cook at home.

## Features
- Dashboard with expiring ingredients, recipe suggestions, and weekly meal plan
- Recipe Library with search and filtering by tag, calories, cooking time, and rating
- Inventory management with expiry tracking and freshness status
- Add / delete recipes and ingredients

## Tech Stack
- React 19 + Vite
- Tailwind CSS
- React Router

## Run Locally
```bash
npm install
npm run dev
```
